What is this? (Beginner guide)

Japanese has two phonetic alphabets that represent the same sounds:

Hiragana (curvy) — for native words & grammar: あa かka さsa

Katakana (angular) — for foreign words (e.g. コーヒー = coffee): アa カka サsa

Each row = same consonant, each column = same vowel. Start with Hiragana!

Voiced (濁音): marks (゛) change the sound — k→g, s→z, t→d, h→b

P-sounds (半濁音): circle (゜) changes h→p
