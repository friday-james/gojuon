What is this?

Japanese has two phonetic alphabets for the same sounds:

Hiragana (curvy) — native words & grammar: あa かka さsa

Katakana (angular) — foreign words (e.g. コーヒー = coffee): アa カka サsa

Each row = same consonant, each column = same vowel. Start with Hiragana!
